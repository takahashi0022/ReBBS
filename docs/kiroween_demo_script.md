## 📝 補足情報

### デモ用のテストデータ準備

デモをスムーズに進めるため、以下を事前に準備：

1. **スレッドを3-5個作成**
   - RSSフィードから自動作成されたもの
   - タイトルが分かりやすいもの

2. **AI投稿を1-2個生成**
   - なんJ語録が含まれているもの
   - 面白い内容のもの

3. **統計データを蓄積**
   - 数回AI投稿を生成して、統計ページにデータを表示

### トラブルシューティング

録画中に問題が発生した場合：

- **AI投稿生成が遅い**: 事前に1回生成して、キャッシュを温めておく
- **エラーが出る**: 事前にテストして、安定した環境を確認
- **画面がカクつく**: 録画ソフトの設定を調整（フレームレート30fps推奨）
